Política de Privacidade da Extensão:
"Copiar Plus Code do Google Maps"

Última atualização: 26 de setembro de 2025

Esta página informa sobre as políticas relativas ao uso de dados da extensão do Google Chrome "Copiar Plus Code do Google Maps" (doravante "Extensão"). A sua privacidade é a nossa prioridade.

1. Coleta e Uso de Dados

A Extensão não coleta, armazena, transmite ou compartilha qualquer tipo de dado pessoal ou de navegação do usuário.

Toda a funcionalidade da Extensão ocorre localmente no seu navegador. O processo é o seguinte:

  1. Ao ser ativada, a Extensão lê o texto que está na sua área de transferência (clipboard).
  2. Ela processa esse texto localmente para converter as coordenadas geográficas em um Plus Code.
  3. O Plus Code gerado é então copiado de volta para a sua área de transferência, substituindo o conteúdo anterior.

As informações lidas da sua área de transferência são usadas unicamente para essa conversão e são descartadas imediatamente após o uso. Em nenhum momento esses dados são salvos ou enviados para qualquer servidor.

2. Permissões Requeridas

Para executar sua função, a Extensão solicita as seguintes permissões. Explicamos abaixo o motivo de cada uma:

  • clipboardRead: Essencial para a função principal. Permite que a Extensão leia as coordenadas geográficas que você copiou para a área de transferência.
  • clipboardWrite: Permite que a Extensão copie o Plus Code resultante para a sua área de transferência, disponibilizando-o para você.
  • scripting e activeTab: São necessárias para exibir a pequena notificação de sucesso ("Toast") na aba ativa, confirmando que o código foi copiado.
  • host_permissions: [""]: Esta permissão é necessária para que a notificação de confirmação ("toast") possa ser exibida em qualquer página que você esteja visitando ao usar a extensão. A extensão não lê, não modifica e não coleta nenhuma informação do conteúdo das páginas que você visita.

3. Alterações a esta Política de Privacidade

Esta Política de Privacidade pode ser atualizada ocasionalmente. Todas as alterações serão refletidas nesta página.
